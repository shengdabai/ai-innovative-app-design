# AI Smart Fitness Coach - AI智能私教

基于AI视觉识别和数据分析的个人健康管理平台

## 项目概述

产品定位：通过"饮食监管 + 智能菜单定制 + 数据闭环分析"为用户提供专业的私教级健康指导服务。

商业模式：SaaS订阅制（年费/月费会员）

## 核心功能

### 1. AI饮食监管与分析
- 📷 拍照识别食物
- 📊 营养成分分析
- 📈 TDEE热量计算
- 🎯 目标对比与建议

### 2. 智能菜单定制
- 🍳 基于现有食材生成菜谱
- 🥗 个性化口味偏好
- ⚠️ 过敏源排除
- 📅 一周菜单计划

### 3. 数据闭环分析
- 📉 体重变化曲线
- 🔥 连续打卡记录
- 🏆 成就系统
- 📊 健康数据报告

### 4. AI对话教练
- 💬 24/7 健康咨询
- 🏋️ 运动指导
- 🥘 饮食建议
- 📈 进度跟踪

## 快速开始

### 环境要求
- Node.js >= 18
- PostgreSQL >= 14
- Redis >= 6
- Docker (可选)

### 后端启动
```bash
cd backend
npm install
npm run start:dev
```

### 前端启动
```bash
cd frontend
npm install
npm start
```

### 使用Docker
```bash
docker-compose up
```
